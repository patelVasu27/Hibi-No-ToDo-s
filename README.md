# Hibi-No-ToDo-s

## Introduction

Hibi-No-ToDo-s is a simple, interactive ToDo list application designed to help users manage their daily tasks efficiently. Built with a focus on user experience, the application provides an intuitive interface for adding, editing, deleting, and organizing tasks. The project leverages modern frontend technologies to deliver a responsive and visually appealing task management solution.

## Features

- Add new tasks with ease using a clear input interface.
- Mark tasks as completed or pending to track your progress.
- Edit existing tasks to update details.
- Delete tasks to maintain a clutter-free list.
- Sort and filter tasks based on completion status.
- Responsive design for use on various devices and screen sizes.
- Persistent storage using browser local storage to retain tasks across sessions.
- Light and dark mode support for better accessibility and user comfort.

## Requirements

- Node.js (version 12 or higher)
- npm (Node package manager)
- A modern web browser for running the application

## Usage

1. **Clone the Repository**
   - Run the following command to clone the repository:
     ```bash
     git clone https://github.com/patelVasu27/Hibi-No-ToDo-s.git
     ```
2. **Navigate to the Project Directory**
   - Change your current directory to the project folder:
     ```bash
     cd Hibi-No-ToDo-s
     ```
3. **Install Dependencies**
   - Install the required dependencies using npm:
     ```bash
     npm install
     ```
4. **Start the Application**
   - Launch the development server:
     ```bash
     npm start
     ```
   - Open your web browser and go to `http://localhost:3000` to view the application.
5. **Build for Production**
   - To create a production build, run:
     ```bash
     npm run build
     ```
   - The build output will be available in the `build` directory.
